# Sudoku Solver

This is a web-based **Sudoku Solver** that solves any custom Sudoku puzzle using the **Backtracking Algorithm**. The site allows users to input their Sudoku puzzle, and the system will automatically solve it using an efficient algorithm. Features like "Back" and "Input Disable" enhance the user experience.

## Features

- **Sudoku Solver**: Solves any valid Sudoku puzzle using backtracking.
- **Backtracking Algorithm**: Implements an efficient algorithm for solving puzzles.
- **Input Disable**: Disables input cells after solving the puzzle to prevent further changes.
- **Back Button**: Allows users to undo the previous move.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript

## How to Use

1. Open the website in a browser.
2. Enter the Sudoku puzzle into the grid.
3. Click "Solve" to solve the puzzle.
4. Optionally, click "Back" to undo the previous step.
5. After solving, the input cells will be disabled to prevent changes.

## Setup

Clone the repository to your local machine to run the project:

git clone https://github.com/your-username/sudoku-solver.git
cd sudoku-solver
Open the index.html file in your browser to view and use the Sudoku solver.
